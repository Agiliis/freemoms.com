[FreeMoms.com](https://freemoms69.netlify.app/) é um site ~piada feito para o aniversário de uma amiga~ que democratiza o acesso à mães para os desprovidos. </br> 
Utiliza-se de HTML, CSS e JavaScript.

## Como usá-lo

Para se requisitar sua mãe gratuita, deve-se informar um código de 8 dígitos **no formato _XXXX-XXXX_** </br>
~No momento, o único código disponível é 7425-7655~

> [!WARNING]
> O site não possui suporte para Firefox (ou navegadores não baseados em Chromium)
